# QtGradientColor
The core code of QtGradientColor is from QtDesigner used to generate qss color code like "qlineargradient(spread:pad, x1:0, y1:0, x2:1, y2:0, stop:0 rgba(0, 0, 0, 255), stop:1 rgba(255, 255, 255, 255))"...

# method to compile this project
VS: 
1、qmake 
2、nmake
or
1、qmake
2、qmake -tp vc

linux
1、qmake
2、make
